# vue-slugify

> Slug directive plugin for Vue.js.

## Install

#### Available through npm as `vue-slugify`.

  ``` js
  var VueSlugify = require('vue-slugify')
  Vue.use(VueSlugify)
  ```

## Usage

#### Using the `v-slugify` directive

``` html
<input type="text" v-slugify>
```
