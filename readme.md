# vue-tippy

> Directive wrapper for Tippy.js 0.11

## Install

#### Available through npm as `vue-tippy`.

  ``` js
  var vueTippyy = require('vue-tippy')
  Vue.use(vueTippy)
  ```

## Usage

#### Using the `v-tippy` directive

``` html
 <button v-tippy> My Button! </button>
```
